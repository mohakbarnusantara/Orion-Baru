1. penjelasan apa itu version control

Version Control adalah sistem yang mencatat setiap perubahan pada file kode. Fungsinya agar kamu bisa melihat riwayat perubahan dan kembali ke versi sebelumnya jika terjadi error atau kesalahan ketik.

2. Perbedaan git dan github

Git: Software/alat yang dipasang di komputer kamu untuk mengelola riwayat perubahan kode (bekerja secara offline).

GitHub: Platform online untuk menyimpan proyek Git kamu di internet agar bisa berkolaborasi dengan orang lain (bekerja secara online).

3. Penjelasan Apa Itu HTML

HTML (HyperText Markup Language) adalah bahasa dasar untuk menyusun struktur sebuah halaman web. HTML menentukan bagian mana yang menjadi judul, paragraf, gambar, atau link. Ibarat sebuah bangunan, HTML adalah kerangkanya.
